## Калькулятор Тома

#### Версия Python
python3.8.3
#### Установка виртуального окружения
`python3 -m venv venv`
#### Применение виртуального окружения
`source venv/bin/activate`
#### Установка зависимостей
`pip install -r requirements.txt`

#### Запуск
`uvicorn main:app --reload`
или
`python main.py`
